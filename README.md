# SOS Labo

> This repository presents a portfolio overview of the project. The full source code is not publicly available.

Interactive JavaFX escape game developed as a Human-Computer Interaction academic project.

## Overview

SOS Labo is a 2D escape game where the player explores a laboratory, solves mini-games, collects keys and unlocks the final exit.

The project focuses on user interaction, scene navigation, keyboard-based controls and a structured MVC architecture.

## Main Features

- Keyboard-based character control
- Multiple game scenes
- Mini-games and final puzzle
- Inventory system
- Interactive objects and visual feedback
- MVC-based architecture

## Technologies

- Java
- JavaFX
- MVC architecture
- UML
- Human-Computer Interaction

## Status
Academic Human-Computer Interaction project.
